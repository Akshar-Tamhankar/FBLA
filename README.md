# FBLA Chapter Hub – Mobile App

## Overview
FBLA Chapter Hub is a mobile application designed to support FBLA members by providing centralized access to chapter information, events, announcements, resources, and social media platforms. The app improves communication, engagement, and organization within the chapter.

This project was developed for the FBLA Mobile Application Development competitive event.

---

## Features

### Calendar and Events
- View upcoming chapter meetings, competitions, and deadlines
- Monthly calendar layout
- Event reminders and detailed descriptions

### Announcements and News
- Chapter updates and important notices
- Competition and leadership information
- Organized news feed

### Resources
- Access to handbooks, templates, and forms
- Quick links to official FBLA materials
- Categorized documents

### Member Profile
- Member information and activity statistics
- Preferences and notification settings
- Achievement badges

### Social Media Integration
- Direct access to chapter, state, and national FBLA platforms
- Dedicated Chapter Social Hub
- In-app browsing and sharing functionality

---

## Installation and Setup

### Requirements
- macOS with Xcode installed
- iOS Simulator or physical iPhone

### Steps
1. Clone or download this repository
2. Open `FBLA.xcodeproj` in Xcode
3. Select an iPhone simulator
4. Press `Cmd + R` to build and run
